User Directory

stephan-haak.com/user-directory

Die Grundlage für jede CRM: ein interaktives Verzeichnis zur Verwaltung von Nutzerdaten, umgesetzt mit React und TypeScript. Mit React Router wurden mehrere Seiten realisiert, während eigene Custom Hooks wiederverwendbare Logik ermöglichen. Die Daten werden im Local Storage gespeichert und das Styling erfolgt mit SCSS. Besondere Herausforderung war das Zusammenspiel von Routing, Datenhaltung und Wiederverwendbarkeit der Komponenten.
